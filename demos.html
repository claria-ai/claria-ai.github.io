See Claria in Action

Short walkthroughs of the features that matter most to clinicians.

AI-Assisted Clinical Documentation

Ask the AI to build a developmental history, draft report sections, or analyze assessment results — all from documents already in the client record.

Voice Memos

Record session notes with live transcription powered by local Whisper inference. Review and edit before saving.

Client Records

Navigate to a client and see their files, records, and documents — all stored securely in your own AWS S3 bucket.

AWS Infrastructure

See the full provisioning status of your HIPAA-compliant AWS resources: encryption, access control, audit trails, and versioning.

Infrastructure Chat

Ask questions about your AWS security configuration in plain English. The AI explains your encryption, access controls, and compliance status.

File Version History

Browse previous versions of any document and compare changes with a side-by-side diff. S3 versioning means nothing is ever lost.

Preferences

Configure prompts, transcription models, and your preferred AI model. Expand each section to customize Claria to your workflow.
